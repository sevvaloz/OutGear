
# OutGear

- An e-commerce mobile application.
- Sells outdoor products such as backpacks, bottle & mugs, hats, tents.

## Technologies

- MVVM
- Live Data
- Hilt
- Coroutines
- Navigation
- Retrofit
- Room Database
- Glide
- Chucker
- Firebase Auth
- Firebase Database
- Bottom Navigation
- Lottie
